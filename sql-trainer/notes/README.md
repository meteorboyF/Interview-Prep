# SQL Study Notes — HR Dataset Edition

Deep study notes that teach **how to think like an engineer**, not just memorize syntax.
Every example runs against the canonical HR training dataset (10 tables, ~25,000 rows) —
paste any query into the [SQL Trainer](https://meteorboyf.github.io/Interview-Prep/sql-trainer/)
playground and it returns exactly the results printed here (every number in these notes
was executed and verified against the real database).

Each chapter follows the same contract per command: definition → syntax → when to use →
when NOT to use → why it exists → benefits → drawbacks → real-world example →
step-by-step execution → comparisons → decision trees → optimization → common mistakes →
interview notes → exam notes → practical notes → memory tricks → summary table →
practice questions → challenge problem.
